# 🏢 LG CNS 회수확인서 프로그램

엑셀 파일을 업로드하여 자동으로 회수확인서를 생성하는 웹 애플리케이션입니다.

## ✨ 주요 기능

- 📊 엑셀 파일 자동 파싱
- 📋 회수확인서 자동 생성
- 🖨️ 인쇄 최적화
- 📱 반응형 디자인
- ⏰ 실시간 날짜 동기화

## 🚀 사용 방법

1. [웹사이트](https://[사용자명].github.io/lg-cns-receipt-generator/) 접속
2. 엑셀 파일 업로드
3. "회수증 생성" 버튼 클릭
4. 생성된 회수증 확인 및 인쇄

## 📋 엑셀 파일 형식

| 자산번호 | 회수코드 | 자산분류 | 모델명 | ... | 반납자 | 연락처 |
|---------|---------|---------|--------|-----|-------|-------|
| 12345   | RC001   | 노트북   | LG그램  | ... | 홍길동 | 010-1234-5678 |
